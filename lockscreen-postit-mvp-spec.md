# 잠금화면 포스트잇 앱 — MVP 스펙

> working name: **구깃 (Googit)** — 종이 구기는 그 느낌. (이름은 미정, placeholder)

---

## 한 줄 요약

머릿속 임시 메모를 잠금화면에 띄워두고, 다 한 건 종이처럼 **구겨서** 버리는 앱.
**추가하기**와 **구기기**, 딱 두 가지만 한다.

---

## 핵심 원칙 (이거 어기면 앱 정체성이 무너짐)

- **수정 없음.** 틀리면 고치는 게 아니라 구기고 새로 추가한다.
- **backlog 아님.** 영구 보관소가 아니라 임시 공간. 쌓이면 구기라는 신호.
- **입력은 미친듯이 빠르게.** 이 앱의 유일한 제품력. 화면 이동·저장 버튼·확인창 전부 금지.

---

## 스택 / 타깃

- **iOS 17.0+** — interactive widget(App Intents in widget)이 필요해서 17 미만은 불가
- **SwiftUI + WidgetKit**
- **로컬 전용** — 계정 / 로그인 / 클라우드 / 네트워크 전부 없음 (오프라인 앱)
- **App ↔ Widget 데이터 공유**: App Group + 공유 컨테이너 (예: `group.com.{you}.googit`)

---

## 데이터 모델

```
Note
  id:        UUID
  text:      String   // 최대 40자 (입력 단계에서 강제)
  createdAt: Date
```

- 정렬: `createdAt` 내림차순 (**최신이 위**)
- 삭제 = 구기기. `isDone` 같은 플래그 없음. 구기면 그냥 영구 삭제.
- 저장: App Group 공유 저장소(UserDefaults suite 또는 단일 JSON 파일). DB 안 써도 됨, 양이 적으니까.

## 앱 — 화면 1개 (그게 전부)

레이아웃 (위 → 아래):

1. **포스트잇 벽** — 메모들이 실제 포스트잇처럼 주르륵. 최신이 맨 위. 스크롤 가능.
2. **하단 입력 바** — iMessage식 채팅 바. 화면 하단 고정, 키보드 따라 올라옴.

동작:

- **앱 열 때**: 자동 포커스 **OFF**. 키보드 안 올라오고 벽이 먼저 보임. (추가하려면 하단 바 탭)
- **추가**: 입력 바에 타이핑 → 엔터 → 새 포스트잇이 **벽 맨 위에 pop** 으로 추가. 입력창 비워지고 그대로 대기.
- **글자 수 제한**: 40자. 넘으면 입력 자체가 막힘 (남은 글자 수 살짝 보여줘도 좋음).
- **구기기**: 포스트잇 탭 → 구겨지는 애니메이션 + 던지기 + 종이 소리 → 삭제.
  - 구길 때 **햅틱** 추가 (`UIImpactFeedbackGenerator`, medium). 손맛 핵심.
- 추가/구기기 후 항상 `WidgetCenter.shared.reloadAllTimelines()` 호출해서 잠금화면 동기화.

## 잠금화면 위젯

- 종류: **rectangular** lock screen widget (시계 아래)
- 내용: **최신 3개**를 한 번에 표시 (공간이 3줄이라)
- **탭 = 다음 페이지** — 3개 넘으면 탭할 때마다 다음 3개로 넘어가고, 끝까지 가면 처음으로 순환
  - 구현: 위젯에 **App Intent** (`CyclePageIntent`) 연결 → 공유 저장소의 `pageIndex` +1 → `reloadAllTimelines()`
- **읽기 전용.** 잠금화면에서는 추가도 구기기도 안 됨 (위젯은 자유 애니메이션·텍스트 입력 불가). 구기기는 무조건 앱 안에서만.

---

## 기술 주의사항 (Claude Code가 꼭 지킬 것)

- 위젯은 스냅샷이라 **구기기 애니메이션을 못 그림.** 구기기 = 앱 전용. 위젯에서 시도하지 말 것.
- 앱과 위젯이 **같은 데이터를 보려면 반드시 App Group 공유 컨테이너**를 써야 함. 앱 전용 저장소 쓰면 위젯이 빈 화면 됨.
- 데이터 바뀔 때마다(추가/구기기/페이지 순환) `reloadAllTimelines()` 안 부르면 잠금화면이 안 갱신됨.
- 탭 순환은 iOS 17 interactive widget + App Intent 패턴. `Button(intent:)` 사용.
- 구기기 애니메이션은 SwiftUI `withAnimation` + transform(scale/rotation) 조합으로. 종이 소리는 짧은 효과음 에셋 or 합성음.

## MVP에서 **빼는 것** (gold-plating 금지)

- 메모 수정 / 상세 보기
- 폴더 · 카테고리 · 태그
- 알림 / 리마인더 / 시간·위치 기반 자동 삭제
- iCloud 동기화 · 계정 · 여러 기기
- 위젯 여러 종류 (circular, 홈화면 등) — rectangular 하나만
- 테마 / 폰트 커스텀
- 정렬 옵션 — 무조건 최신순 고정

---

## 빌드 순서 (각 단계 독립적으로 테스트 가능)

1. **데이터 + App Group** — Note 모델, 공유 저장소 읽기/쓰기, 추가·삭제 함수
2. **앱 단일 화면** — 포스트잇 벽(최신순) + 하단 입력 바 + 추가(40자 제한, pop)
3. **구기기** — 탭 → 구김 애니메이션 + 햅틱 + 소리 + 삭제
4. **위젯** — rectangular, 최신 3개 표시, `reloadAllTimelines()` 연동
5. **탭 순환** — `CyclePageIntent`로 페이지 넘기기 + 순환
